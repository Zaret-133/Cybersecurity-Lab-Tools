# Week 03 - Security Automation with Python

## Overview
This week focused on building automated security tools using Python to detect threats and analyze system activity.

## Skills Developed
- Python scripting for security
- Log parsing and analysis
- Automation of repetitive security tasks
- JSON report generation

## Projects & Artifacts
- port_check.py → Script to scan open ports across multiple systems
- brute_detector.py → Tool to detect failed login attempts from logs
- system_auditor.py → Script to identify unauthorized processes
- incident_response.py → Automated log analysis and threat reporting
- threat_report.json → Structured threat intelligence output

## Tools Used
- Python (socket, subprocess, json)
- Linux CLI

## Key Takeaways
- Built reusable automation tools for security operations
- Learned how to parse logs and extract meaningful data
- Transformed manual analysis into scalable scripts
