# 🛡️ Week 10 — The Defender (DFIR)

---

# 📌 Overview

This week focused on Digital Forensics and Incident Response (DFIR). The objective was to investigate cyber incidents, preserve forensic evidence, recover deleted artifacts, and reconstruct attack timelines using professional DFIR workflows and SIEM platforms.

---

# 🎯 Lab Objective

Learn how incident responders and forensic analysts investigate compromised systems, preserve evidence integrity, analyze logs, recover deleted files, and reconstruct attacker activity.

---

# 🧠 Skills Developed

- Incident Response Lifecycle (PICERL)
- Live System Triage
- Cryptographic Hashing
- Chain of Custody Procedures
- Memory Forensics
- Disk Imaging & Recovery
- SIEM Navigation
- Log Correlation
- IOC Identification
- Attack Timeline Reconstruction
